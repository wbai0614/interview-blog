# Linux

Linux is a popular operating system used in servers, DevOps, and software engineering interviews. Understanding core commands and concepts is essential.

## File System Navigation

- `ls` - list files and directories  
- `cd` - change directory  
- `pwd` - print working directory  
- `mkdir` - create directory  
- `rm` - remove file or directory  

```bash
cd /var/log
ls -la
mkdir new_folder
rm old_file.txt
