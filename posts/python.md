# Python: List vs Tuple

## Key Differences

| Feature | List | Tuple |
|------|------|-------|
| Mutability | ✅ Mutable | ❌ Immutable |
| Performance | Slower | Faster |
| Use case | Dynamic data | Fixed data |

## When to Use Tuple
- Dictionary keys
- Constants
- Return multiple values
