# Interview Blog

Markdown-powered interview notes blog built with **Node.js**, **Express**, and **EJS**, deployed on **Google Cloud Run**.

**Live demo:** [Interview Blog](https://interview-blog-133059075456.us-central1.run.app/)

---

## Features

- Markdown-based posts with **frontmatter** (title, tags, description, featured)  
- Dynamic homepage with **search** and **tag filters**  
- Featured posts section  
- Individual post pages with **reading progress bar**  
- Syntax highlighting ready  
- Fully containerized for deployment to **Google Cloud Run**  

---

## Deploy using PowerShell script
.\deploy-cloudrun.ps1
